# E-Commerce Website

A Ecommerce Website made with React.js Framework.



## Run Locally

Clone the project

```bash
  git clone https://github.com/anirban-srmu/react-e-commerce
```

Go to the project directory

```bash
  cd react-e-commerce-main
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
